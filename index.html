PGNTA: 17 DOF Humanoid Arduino Robot

Welcome to the project page of PGNTA, a humanoid robot built during my first-year summer break.

Project Overview

PGNTA is a 17 Degree of Freedom (DOF) humanoid robot powered by an ESP32 microcontroller. Built primarily from wooden plywood, it combines servo motors, sensors, and custom-designed joints to create a functional bipedal structure.

Key Features

  • 17 DOF using MG995 and SG90 servo motors
  • ESP32 microcontroller for main control
  • Custom wooden plywood frame
  • Ultrasonic sensor and MPU6050 for balance and object detection
  • Arduino Cloud integration for remote monitoring

For complete project details, please visit the GitHub repository.
